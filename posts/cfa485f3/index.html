数学归纳法

法一:

  • 验证n=1时,命题正确;
  • 设n=k时,假设结论;
  • 验证n=k+1时,假设依然成立.

法二:

  • 验证n=1,n=2;
  • 设n<k时,成立;
  • 验证n=k时,成立.

如果只有一个需要归纳假设, 用法一; 如果有两个需要假设, 用法二.

特征值和特征向量

  • 实对称矩阵的特征值的和等于对角线元素和.

例:

  • 特征值等于0所对应的特征向量是基础解系

方程组

  • 齐次有解(AX = 0):

    • 必有0解
    • 若r(A)<n,即约束条件少于未知数,则有非0解

  • 非齐次有解(AX = b):

    • 只有r(A) = r(A+),即矩阵A的秩等于增广矩阵(加上b)的秩,才会有解
    • r(A) = r(A+) < n,约束条件少了,有无穷多解.
    • r(A) = r(A+) = n,约束条件刚刚好,只有唯一解.

  • 约束条件多了不一定有用,但少了一定会造成解变多.

  • A^(n+1)X=0的解必是A^nX=0的解

A'x = 0的解
A的伴随的齐次方程组的通解

  1. 求A的秩, 若r(A) = n-1, Ax = 0必有非0解, 且|A| = 0.
  2. 由1得r(A') = 1, A'A = |A|E = 0. 注意此处 A'A = |A|E 并非通过A的逆得出, 这是个固定性质.
  3. 于是A的列向量就是 A'x = 0 的解.
  4. 然后从A选出n-r(A')即n-1个不相关的列向量即是 A'x = 0 的基础解系.

向量


列向量构成的矩阵一定秩为1

向量组

  • 低维无关,高维必无关(3个位置变量的方程组只有0解,添加一个变量还是只有0解)
  • m个n维向量,若m>n,即约束条件多于未知量,必有非零解,所以线性相关.

秩

  • 若向量组A可由B线性表出,r(A)≤r(B).
  • r(AB) ≤ min( r(A) , r(B) ),若A可逆,则r(AB) = r(B) = r(BA).
  • 基础解系的个数即矩阵自由变量的个数,为n-r(A).
  • 相似矩阵的秩也相同
  • r(A的伴随*A) = r(A)

    若AB = 0 ➡ r(A)+r(B)≤n

    证明:
    若AB = 0, 则B的列向量是齐次方程组AX=0的解, 设r(A) = r, 则r(B) ≤ n-r, 所以r(A) + r(B) ≤ n.

等价

矩阵A经过有限次初等变换得到B,就叫A与B等价.

  • A等价与B ↔ r(A)=r(B).

逆

  • (A+B)的逆没有化简公式

相似矩阵

  • 相似矩阵的秩、行列式值、迹、特征值都相同.
  • 三角矩阵(主对角线)的特征值就是主对角线上元素.
  • 对角矩阵特征值就是对角线元素

相似对角化

  • A~Λ ↔ A有n个线性无关的特征向量 ↔ λ_i是A的n重特征值,而λ_i有n个线性无关的特征向量
    ↔ r(λ_i*E - A) = n-n_i,λ_i为n_i重特征值
  • A有n个不同的特征值 -> A~Λ
