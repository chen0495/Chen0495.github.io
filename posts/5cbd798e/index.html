直线和线段


组成了一条穿越x₁和x₂的直线，当0 <= θ <= 1时，组成一条线段。

仿射集合


也就是说在n维空间里的集合 C ，通过 它里面的任意两点 组成的直线仍然属于集合 C ，那么就说 C 是仿射的。
推论:
C 是一个仿射集，取其中任意点x₁、x₂、x₃… ,若有θ₁+θ₂+θ₃+…=1 ,那么y=θ₁x₁+θ₂x₂+θ₃x₃….仍然在集合 C 中。

凸集

我们定义一个集合是凸的，当且仅当 任意x,y∈C 且 θ∈R,0≤θ≤1,
θx+(1−θ)y∈C
也就是一个集合C是凸集，当且仅当C中任意两点间的线段仍然属于C
其中点θx+(1−θ)y被称作点集x,y的凸性组合
